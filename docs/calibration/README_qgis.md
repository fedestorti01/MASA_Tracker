# How to obtain georeferenced maps for calibration

* Install QGIS (https://qgis.org/en/site/forusers/download.html)
* Install Tile+ plugin already provided in this folder as tile_plus.zip, following this tutorial (https://geographicalanalysis.com/gis-blog/download-tile-plus-plugin-qgis/)
* Decide the area you want to export
* Start QGIS
* Follow the video tutorial export_tiff_qgis.mp4, note that in the video IGM95 is selected as the coordinate reference system but you can choose your preferred one. Google Maps uses WGS84, select this one if you want to precisely display the coordinates produced on Google Maps
